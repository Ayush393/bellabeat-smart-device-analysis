## Bellabeat Smart Device Behavioral Analysis
Marketing analytics case study analyzing Fitbit smart device usage using Python (Pandas, Matplotlib, Seaborn).

# Business Problem

Analyze Fitbit smart device usage data to uncover trends in activity and sleep behavior and provide marketing recommendations for Bellabeat's Leaf wellness tracker.

# Tools Used

Python

Pandas

NumPy

Matplotlib

Seaborn

Google Colab

# Analysis Performed

Data cleaning and preprocessing

Date transformation

Feature engineering (TotalActiveMinutes, SleepHours)

Dataset merging

Descriptive statistics

Correlation analysis

Weekday behavioral trends

User-level aggregation

# Key Insights

Average daily steps ~7,000–8,000

Strong correlation between steps and calories

Many users average under 7 hours sleep

Behavioral routines structured around weekdays

# Marketing Recommendations

Emphasize sleep optimization messaging

Target moderately active users with habit-based goals

Use weekday-driven engagement campaigns
